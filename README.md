Blender 3D: Ricostruzione di una stanza
Progetto di grafica 3D realizzato in Blender, con l’obiettivo di ricreare una stanza generata a partire dalla mia matricola universitaria (354760).
**Attività svolta:**
- Creazione e modellazione degli oggetti  
- Applicazione di materiali e texture  
- Impostazione dell’illuminazione della scena  

**Strumenti utilizzati:**
- Blender (versioni 4.x e successivamente la 5.x)  
- Motore di rendering: Cycles / Eevee
